## Use Case
As a marketing manager, I need to track my campaigns and monitor spending across all channels, adjusting spend as needed.

## Technology
- Tech 1: Budget Manager (Approved Budget - By Quarter, By Channel, By Region, By Campaign)
- Tech 2: Expense Calculator (Total Spend, Campaign Spend - By Quarter, By Channel, By Region, By Campaign)
- Tech 3: Utilization Calculator (Total Spend vs Approved Budget)
- Tech 4: Impression/Conversion Counter (CPM, CPC, CPA)
- Tech 5: ROAS Calculator (rate of ad spend averages)

## Design Notes
- AI Prompt: Create an internal dashboard to track marketing metrics for global marketing campaigns. Marketing managers need insights to cost per imporession, cost per click, cost per acquistion, and average rate of ad spend per social media channels fro each campaign. These metrics will allow them to manage quarterly marketing budgets. Color code any variants over approved budgets. Seed data for the dashboard.
